Chat Server - Web Client Configuration

📋 What You Need

To run the web client, you'll need:

  • Node.js - For the WebSocket proxy
  • csServer2 - Running on port 8888
  • Web Browser - Chrome, Firefox, Edge, Safari
  • Terminal/PowerShell - To start the proxy

If you don't have Node.js, download it from nodejs.org

🎮 How It Works

The web client uses a WebSocket-to-TCP proxy:

  • Browser → WebSocket connection
  • Proxy → Bridges the gap
  • csServer2 → TCP socket server

This allows browsers to connect to the TCP server

🚀 Quick Setup

1Start csServer2

Build and run csServer2 in Visual Studio or from the terminal:

dotnet run --project csServer2/csServer2.csproj
2Install Dependencies

Open PowerShell in the csWebClient folder and run:

npm install

This installs the WebSocket library needed for the proxy.

3Start the Proxy Server

In the same PowerShell window, run:

npm start

You should see: [PROXY] Waiting for connections...

Keep this terminal open! The proxy must be running while you use the web client.

4Open the Login Page

Open your browser and navigate to:

file:///path/to/csWebClient/Login.html

Or use Python's built-in server:

python -m http.server 8000
Then open: http://localhost:8000/Login.html
5Login

Enter your username and click Login. Enjoy the Battle Arena!

⚠️ Important: Both csServer2 AND the proxy (npm start) must be running for the web client to work. The proxy bridges your browser (which can't connect to TCP) with the server.
